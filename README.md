# SpryDev

SpryDev provides a [Vagrant](http://vagrantup.com) based on
[Ubuntu 14.04.1 LTS (Trusty Tahr)](http://releases.ubuntu.com/14.04/) with
[Ansible](http://ansible.com) and [Docker](http://docker.com).

The goal of SpryDev is to provide a minimal, consistent, cross platfrom
development environment for projects using Ansible and Docker for configuration
management and deployment.

## Prerequisites

* [VirtualBox](http://virtualbox.org)
* [Vagrant](http://vagrantup.com)
* [Vagrant Hostmanager](https://github.com/smdahlen/vagrant-hostmanager) Plugin
